This folder contains the native applications that can be run from an accessory of the AOZ editor.
The application should be in a "portable" format if possible, to avoid installation and dependencies.

Ce dossier contient les applications natives qui peuvent être exécutées depuis un accessoire de l'éditeur AOZ.
L'application doit être dans un format "portable" si possible, afin d'éviter une installation et des dépendances.
